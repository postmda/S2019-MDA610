Simple HTML

This is a simple HTML file.

It is used to illustrate the Document Object Model (DOM), which describes the structure of an HTML document and is at the heart of interacting with the browser.

Conceptually, the DOM is a tree. The document is the root node with the html element as a child. The html element (document.documentElement), in turn, has two children: the head element (document.head) and the body element (document.body).

Every node in the DOM tree is an instance of the Node class. Each Node object has a parentNode property and a childNodes property.

Every node also has the properties nodeName and nodeType. The types of nodes primarily used are Node.ELEMENT_NODE for HTML elements and Node.TEXT_NODE for text contents within HTML elements.

The DOM provides “get” methods to quickly locate specific HTML elements. The document methods querySelector and querySelectorAll enable you to use CSS selectors. Each element has two properties, textContent and innerHTML that allow you to access and change the element's content.
