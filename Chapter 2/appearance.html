MDA 610 Data Management and Mining

Now Open for Spring Registration

Course Description

The goal of this 3-credit course is to provide a comprehensive introduction to the principles and tools for managing and mining data, covering database management, data retrieval, data preprocessing, data analysis and mining.

What You'll Learn

  • Web development
  • Database management
  • Data visulization
  • Data analysis
  • Data mining

Programming Languages

  • HTML5
  • CSS3
  • JavaScript
  • MySql
  • Python

Why Register?

You should register if you want to understand how data can be explored to uncover hidden informaiton, learn the data mining and management techniques to effectively improve business performance, or get into the data science field.

This is a 3-credit graduate-level course open to all majors.
